# Letter Lab

A touch-first web app for number and letter recognition practice on iPhone, Kindle Fire tablets, and desktop browsers.

## Features

- Letter case matching, number quantity matching from 1-10, and mixed practice modes
- Easy and challenge answer sets
- Large mobile-friendly answer cards
- Built-in speech prompt through the browser's speech synthesis
- Finger tracing canvas for each symbol
- Daily progress saved in the browser
- Offline caching after the first load

## Run Locally

From this folder:

```powershell
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

For an iPhone or Kindle Fire on the same Wi-Fi network, replace `localhost` with your computer's local IP address.

## Install On Device

- iPhone Safari: open the site, tap Share, then tap Add to Home Screen.
- Kindle Fire Silk: open the site, use the browser menu, then choose Add to Home Screen if available.
